## Build scripts

Building bash script
```shell
$ ./build.sh
```

Running bash script
```shell
$ ./run.sh
```

Running nushell script
```shell
$ nu run.nu
```
or
```shell
$ ./run.nu
```

Install the npc c library
```shell
$ ./install.nu
```
or
```shell
$ nu install.nu
```
